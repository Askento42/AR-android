# AR Menu (готовый прототип)
Структура:
- `index.html` — готовая страница.
- `assets/models/soup_v2.glb` — положите сюда вашу модель (имя важно).
- `assets/posters/soup_poster.jpg` — постер (можно удалить или заменить).

Заливка на GitHub Pages:
1) Создайте публичный репозиторий `ar-menu` (любое имя).
2) Загрузите весь содержимое этой папки в корень репозитория.
3) В Settings → Pages включите публикацию: Branch `main` / root.
4) Откройте URL вида `https://<user>.github.io/ar-menu/`.

Если текстура не видна только на странице (в editor видно) — чаще всего кэш.
В проекте уже есть обход (`?v=DEV`). При необходимости переименуйте `soup_v2.glb`.
